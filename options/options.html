🧠 Output Learning MDX

Il nuovo export genera sempre un solo file .mdx completo con tutte le viste: trascrizione integrale, studio guidato, sintesi rapida, mappa concettuale e asset didattici.

Questa scelta non cambia il contenuto generato. Il file include sempre tutte le viste e decide solo quale pannello aprire per primo.

MDX permette struttura più ricca, blocchi HTML/MDX, navigazione interna e future estensioni interattive.

💾 Salvataggio File

I file .mdx vengono scaricati nella cartella LearningHub/ dentro la tua cartella Download. Opzionalmente puoi collegare direttamente il tuo vault Obsidian.

Usa File System Access API: il browser chiede il permesso una sola volta. I file vengono poi salvati automaticamente nella cartella scelta.

☁️ Cloud Sync (Notion / GitHub)

Sincronizza automaticamente gli appunti nel Cloud ogni volta che estrai un video.

Crea un'integrazione interna in Notion e convidivi il tuo database con essa.

Il token deve avere i permessi per scrivere contenuti nel repository ("repo" scope).

⌨️ Scorciatoie Tastiera

Le scorciatoie permettono di agire senza aprire il popup. Chrome gestisce direttamente le combinazioni di tasti — puoi modificarle dalla pagina dedicata.

Aggiungi video alla coda Su qualsiasi video YouTube, aggiunge il video alla coda senza aprire il popup né avviare l'analisi AI.
⌘ Cmd + ⇧ Shift + P

⚠️ Chrome non permette alle estensioni di cambiare le scorciatoie direttamente — devi farlo dalla pagina di sistema.

📥 Auto-Queue

TubeBrain può controllare automaticamente i feed dei creator che segui e accodare i nuovi video pubblicati dopo il follow.

Il controllo avviene in background anche quando il browser è aperto. I video vengono accodati e restano in attesa di estrazione. Creator con priorità ⚡ vengono estratti immediatamente; gli altri restano in coda.

L'AI valuterà (Screening) i nuovi video in background per capire se corrispondono ai tuoi interessi, assegnando un Relevance Score da 1 a 10.

ℹ️ Come funziona

  1. Apri un video YouTube
  2. Clicca l'icona 📚 TubeBrain nella barra del browser
  3. Scegli la modalità iniziale del file e premi Genera Learning MDX
  4. Il provider AI selezionato analizza la trascrizione e genera il documento multi-livello
  5. Salva il file .mdx e trovalo nell'Archivio della Dashboard
  6. Aggiungi creator dalla Dashboard per monitorare i loro nuovi video
