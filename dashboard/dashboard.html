🚀 Inizio Rapido

TubeBrain trasforma qualsiasi video YouTube in un documento Markdown strutturato, pronto per Obsidian o qualsiasi editor di note. Ecco come iniziare in 3 minuti.

1
Configura un provider AI

Puoi usare Google AI Studio per Gemini oppure la tua API key OpenAI. Il provider attivo si gestisce nelle Impostazioni.

2
Inserisci la chiave nelle Impostazioni

Clicca ⚙️ Impostazioni in sidebar → incolla la chiave nel campo Gemini API Key → premi Test per verificarla → Salva.

3
Apri un video YouTube e clicca l'icona 📚

Naviga su un video YouTube → clicca l'icona TubeBrain nella barra del browser → premi ✨ Genera Learning MDX. Il file viene salvato automaticamente in LearningHub/.

✅ Fatto! Il tuo primo file Markdown è nella cartella Download, dentro LearningHub/{NomeCreator}/{Anno}/

🔑 Come Ottenere le API Gratuite

TubeBrain usa due API di Google, entrambe gratuite per uso personale. Ecco come ottenerle passo per passo.

Obbligatoria

🤖 Gemini API Key — Google AI Studio

Usata per: generare riepiloghi MD, chat con l'archivio, ricerca semantica, analisi articoli web, alert topic, estrazione tag.

Nota provider: Gemini resta il percorso gratuito più semplice; OpenAI è disponibile via API con runtime attuale forzato su gpt-5.4-mini.

1
Vai su Google AI Studio

Apri aistudio.google.com/apikey — accedi con il tuo account Google (lo stesso di Gmail va bene).

2
Crea un nuovo progetto (se richiesto)

Se è la prima volta, Google ti chiede di creare un progetto. Dai un nome a piacere (es. "TubeBrain") e conferma.

3
Clicca "Create API key"

Nella pagina API Keys, clicca il pulsante blu "Create API key". Scegli il progetto appena creato (o quello esistente) → "Create API key in existing project".

4
Copia la chiave

La chiave inizia con AIza.... Clicca l'icona 📋 per copiarla. Non condividerla mai pubblicamente.

5
Incollala nelle Impostazioni di TubeBrain

Sidebar → ⚙️ Impostazioni → campo "Gemini API Key" → incolla → clicca Test per verificare → Salva Impostazioni.

✅ Fatto! Con questa sola chiave puoi già usare tutte le funzioni principali di TubeBrain: riepiloghi, chat, ricerca semantica.
Opzionale

📺 YouTube Data API v3 — Google Cloud Console

Usata per: avatar del canale, numero iscritti, numero video pubblicati, paese del creator. Senza questa chiave, le card creator mostrano solo l'iniziale del nome.

Limite gratuito: 10.000 unità/giorno (ogni richiesta dati creator = ~1-5 unità). Praticamente illimitato per uso personale.

1
Apri Google Cloud Console

Vai su console.cloud.google.com → accedi con lo stesso account Google usato per AI Studio.

2
Seleziona (o crea) lo stesso progetto

In alto a sinistra, clicca sul selettore progetto → seleziona il progetto che hai creato in AI Studio (es. "TubeBrain"). Se non lo vedi, cerca per nome.

3
Abilita YouTube Data API v3

Vai su APIs & Services → Library → cerca "YouTube Data API v3" → clicca sul risultato → clicca "Enable" (o "Abilita").

4
Crea o riutilizza una API Key

Vai su APIs & Services → Credentials. Se hai già creato una chiave API per questo progetto (es. la stessa usata per Gemini), puoi usare la stessa chiave — funzionerà per entrambe le API. Altrimenti clicca "Create Credentials" → "API key".

5
Incollala nelle Impostazioni

Sidebar → ⚙️ Impostazioni → campo "YouTube Data API v3 Key" → incolla → clicca Test → Salva. Puoi incollare la stessa chiave Gemini se il progetto ha entrambe le API attive.

💡 Suggerimento: Usa la stessa API key per entrambi i campi — è il metodo più semplice. Assicurati solo che il progetto Google Cloud abbia sia "Generative Language API" (Gemini) che "YouTube Data API v3" abilitate.
⚠️ Mai condividere le API key. Non incollarle in chat, non pubblicarle su GitHub, non inviarle in email. Se per errore le esponi pubblicamente, revocale subito dalla console Google e creane di nuove.
Nessuna chiave

📸 Instagram — Analisi senza API aggiuntive

Per analizzare post, reel e profili Instagram non serve nessuna API aggiuntiva. TubeBrain usa la Gemini API Key già configurata per analizzare il testo visibile nella pagina.

Come funziona: apri la pagina Instagram nel browser (post, reel o profilo) → clicca l'icona 📚 TubeBrain → clicca 📸 Analizza. L'estensione legge la caption, gli hashtag e il testo visibile nella pagina, poi Gemini genera il riepilogo.

⚠️ Limitazioni Instagram: i Reel e i video non hanno trascrizione audio (a differenza di YouTube). L'analisi si basa solo sul testo scritto (caption, hashtag, bio). Per risultati migliori, assicurati di essere loggato su Instagram prima di analizzare.

🕐 La Coda — Salva Ora, Estrai Dopo

Stai guardando un video interessante ma non hai tempo per aspettare l'analisi AI? Aggiungilo alla coda: viene salvato nell'archivio come "Da estrarre" e puoi elaborarlo quando vuoi.

Come aggiungere alla coda

⌨️
Scorciatoia tastiera

⌘ Cmd + ⇧ Shift + P su Mac
Ctrl + Shift + P su Windows/Linux

Funziona senza aprire il popup — puoi farlo mentre guardi il video!

📌
Pulsante nel popup

Apri il popup → clicca 🕐 Coda. Stesso risultato della scorciatoia.

Come estrarre i video in coda

Vai nella tab Archivio → usa il filtro "🕐 Da estrarre" → clicca ✨ Estrai sulla card del video.

L'estrazione avviene completamente in background: TubeBrain apre la pagina YouTube in una tab nascosta, legge la trascrizione, chiama Gemini, salva il file e chiude la tab. Riceverai una notifica Chrome al termine.

⚠️ L'estrazione in background richiede che Chrome rimanga aperto. Non chiudere il browser durante l'elaborazione.

Personalizzare la scorciatoia

Vai su chrome://extensions/shortcuts → trova "TubeBrain" → clicca la matita accanto a "Aggiungi video alla coda" → inserisci la combinazione preferita.

💬 Chat con l'Archivio

La chat ti permette di fare domande in linguaggio naturale su tutti i tuoi riepiloghi salvati. Gemini analizza i documenti più rilevanti e risponde citando le fonti.

🔑 Richiede: Gemini API Key configurata nelle Impostazioni. Non richiede API Key YouTube.

Come funziona (RAG)

Quando invii una domanda, TubeBrain usa un approccio RAG (Retrieval Augmented Generation):

1
Pre-filtraggio keyword

La domanda viene analizzata per trovare le parole chiave e selezionare i riepiloghi più rilevanti dal tuo archivio (fino a 25).

2
Costruzione del contesto

I riepiloghi selezionati vengono inclusi nel prompt (fino a ~380.000 caratteri di testo). Gemini non "conosce" i tuoi documenti — li legge ogni volta in tempo reale.

3
Risposta con citazioni

Gemini risponde basandosi esclusivamente sui tuoi documenti, citando le fonti con [Titolo]. Le fonti usate appaiono come chip cliccabili sotto la risposta.

Esempi di domande utili

Tipo di domandaEsempi
Riepilogo tematico"Cosa ho imparato su Python?" · "Riassumi tutto quello che so di SEO"
Ricerca concetti"Come funziona il sistema di autenticazione OAuth?" · "Cosa dice il mio archivio sui database vettoriali?"
Confronto"Quali sono le differenze tra i tool di automazione citati nei miei video?"
Discovery"Quali argomenti ho studiato di più?" · "Da quali creator ho imparato di più?"
Azione"Dammi un piano di studio basato sui contenuti che ho salvato"
⚠️ La chat risponde solo in base ai tuoi riepiloghi. Se un argomento non è presente nell'archivio, Gemini lo dirà esplicitamente anziché inventare. Per risultati migliori, estrai più riepiloghi.

🏠 Il Feed — Tieni Traccia dei Creator

Il feed aggrega i video più recenti di tutti i creator che segui, come un RSS reader personalizzato. I dati vengono recuperati dal feed XML pubblico di YouTube — senza chiavi API aggiuntive.

Aggiungere un creator

Vai in Creator → incolla l'URL del canale YouTube in uno di questi formati:

https://youtube.com/@nomecreator
https://youtube.com/channel/UCxxxxxxx
https://youtube.com/c/nomecreator

Statistiche canale (opzionale)

Se inserisci una YouTube Data API v3 Key nelle impostazioni, ogni card creator mostrerà: avatar reale, numero iscritti, numero video, paese. Senza chiave, viene usato il fallback (iniziale del nome).

Per ottenere la chiave: Google Cloud Console → abilita YouTube Data API v3 → crea credenziale API Key. Puoi usare la stessa chiave Gemini se il progetto ha entrambe le API attive.

Aggiornamento automatico

Il feed si aggiorna automaticamente ogni 30 minuti. Clicca 🔄 per forzare un aggiornamento immediato. I video già riassunti mostrano il badge ✅.

📄 Il File Markdown Generato

Ogni output principale è un file .mdx compatibile con workflow Markdown avanzati e archivio locale. Include frontmatter YAML e sezioni a 3 livelli per studio, ripasso e replicazione.

Struttura del documento

SezioneContenuto
📋 Riepilogo Generale3-4 paragrafi: di cosa tratta, cosa si impara, a chi è rivolto, risultato finale
🎯 PrerequisitiConoscenze, software e configurazioni necessarie prima di seguire il tutorial
🛠️ Tool e SoftwareTabella di tutti gli strumenti, librerie e servizi menzionati nel video
📚 Capitoli DettagliatiOgni sezione del video con: riepilogo, passaggi eseguiti, codice mostrato
💻 Raccolta CodiceTutto il codice e i comandi del video raccolti in un unico blocco
🔑 Punti Chiave7-10 takeaway specifici e actionable
🔄 BlueprintStruttura narrativa del video replicabile per creare contenuti simili
📎 Risorse e LinkTutti i link, canali, libri e strumenti citati
🤖 Workflow AntigravityIstruzioni operative per replicare il workflow in Google Antigravity

Frontmatter YAML (compatibile Obsidian)

---
title: "Titolo del video"
channel: "Nome Creator"
date: 2026-05-10
tags: [python, tutorial, automazione]
source: "https://youtube.com/watch?v=..."
platform: youtube
---

Lingua del documento

Nelle Impostazioni puoi scegliere: Italiano (sempre), English (sempre), o Automatico (segue la lingua del sistema operativo). La lingua del video non influisce — Gemini traduce automaticamente.

🤖 Workflow Antigravity

La sezione finale di ogni file MD contiene istruzioni operative pronte per Google Antigravity, la piattaforma di Google per agenti AI automatici.

Come usare le istruzioni

1
Apri il file MD generato

Vai alla sezione ## 🤖 Workflow Antigravity in fondo al documento.

2
Copia il "Prompt di Esecuzione Rapida"

Trovi un blocco di testo pronto all'uso, con la descrizione completa del workflow in linguaggio naturale.

3
Incolla in Google Antigravity

Apri antigravity.google → crea un nuovo agente → incolla il prompt → configura i tool indicati nella tabella → avvia.

Struttura YAML del workflow

Il file include anche una definizione YAML con: nome workflow, input richiesti, steps dettagliati (tool, azione, input, output atteso, gestione errori), e variabili che si passano tra steps ({{step_1.output}}).

💡 Le istruzioni Antigravity sono generate automaticamente da Gemini analizzando i passaggi tecnici del video. Più il video è un tutorial pratico con step precisi, più le istruzioni saranno accurate e operative.

💡 Tips & Tricks

⚡
Elabora mentre guardi

Mentre guardi un video interessante, premi ⌘⇧P per metterlo in coda. Poi elabora tutti in batch dalla Dashboard quando sei pronto.

🔍
Cerca nel contenuto

La ricerca in Archivio trova parole dentro il testo dei file MD, non solo nei titoli. Cerca un comando, un tool o un concetto per trovare il video giusto.

🗂️
Integrazione Obsidian

Seleziona la cartella del vault nelle Impostazioni. I file vengono creati con frontmatter YAML e tag automatici, pronti per il graph view di Obsidian.

📊
Monitorare l'apprendimento

La tab Statistiche mostra il grafico dell'attività mensile e i topic più studiati. Utile per tracciare il tuo percorso formativo nel tempo.

🎯
Video senza sottotitoli

Se un video non ha sottotitoli, Gemini analizza comunque il titolo, la descrizione e i metadati. Il riepilogo sarà meno dettagliato ma sempre utile.

🔄
Modello Pro per video complessi

Per tutorial lunghi e densi di contenuto tecnico, passa a Gemini 2.5 Pro nelle impostazioni per un'analisi più accurata (stesso costo: gratuito).

🐛 Problema tecnico? Nella maggior parte dei casi basta ricaricare la tab YouTube con F5 e riaprire il popup. Se il problema persiste, vai su chrome://extensions → 🔄 Aggiorna TubeBrain → F5 su YouTube.
